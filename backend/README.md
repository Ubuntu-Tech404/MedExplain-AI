# Mediclinic AI Backend

AI-powered medical dashboard backend using FastAPI and Llama 3.2 11B.

## Features

- Medical text explanation using Llama 3.2 11B
- Diagnosis interpretation and education
- Lab result analysis and categorization
- Document processing (PDF, DOCX, TXT)
- Health visualization and charts
- Patient data management
- Authentication and authorization

## Installation

1. Create virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```
